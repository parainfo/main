데이터셋, 가이드북 소개

미래차 제조기업의 솔루션 실증 및 자체 데이터 분석 지원을 위해 활용성 높은 현장 데이터를 선별한 데이터셋을 구축하고, 자체 분석능력이 부족한 기업들이 데이터셋을 직접 분석할 수 있도록 분석 가이드북을 제공합니다.
구축한 데이터셋은 한국데이터산업진흥원의 데이터 품질 평가 지표를 활용하여 전문가의 평가와 검증을 통해 데이터의 신뢰성을 확보하였습니다.
품질 지표 정의
완전성 데이터셋의 레코드가 누락이 없거나 반드시 값이 입력되어 있어야 하는 항목이 누락 없이 잘 관리되는지 평가함
유효성 수치형 데이터 값은 유효한 범위에 있는지, 범주형 데이터 값은 유효 값의 목록을 준수하고 있는지 평가함
정확성 메타데이터에 기재된 값과 실제 등록된 값이 정확한지 평가함 (건수, 크기, 항목개수 등)
일관성 동일한 데이터 항목이 동일한 데이터 형식을 갖는지 평가함
유일성 데이터셋의 항목이 유일해야 하나 중복이 있는지 평가함
적시성 가장 최신 데이터를 제공해야 하며 갱신 주기에 따라 정상적으로 갱신되었는지 평가함
활용성 데이터를 활용하는데 있어 법적인 문제가 발생하지 않도록 하기 위하여 개인정보가 비식별화 정책에 따라 비식별화 되어 익명성이 보장되는지 평가함
